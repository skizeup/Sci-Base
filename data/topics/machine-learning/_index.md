# Machine Learning

## Description
Le machine learning (apprentissage automatique) est une branche de l'intelligence artificielle qui permet aux systèmes d'apprendre à partir de données sans être explicitement programmés.

## Sous-domaines couverts
- Apprentissage supervisé
- Apprentissage non supervisé
- Apprentissage par renforcement
- Deep learning
- Transformers & attention
- NLP (Natural Language Processing)
- Computer Vision

## Prérequis recommandés
- Algèbre linéaire
- Probabilités et statistiques
- Programmation Python
