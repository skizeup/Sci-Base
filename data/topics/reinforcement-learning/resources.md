# Ressources — Reinforcement Learning

## Cours en ligne
- [David Silver — RL Course (UCL/DeepMind)](https://www.davidsilver.uk/teaching/) — le cours de référence
- [Stanford CS234 — Reinforcement Learning](https://web.stanford.edu/class/cs234/)
- [Spinning Up in Deep RL (OpenAI)](https://spinningup.openai.com/) — guide pratique avec code

## Livres
- *Reinforcement Learning: An Introduction* — Richard Sutton, Andrew Barto (le livre fondateur, gratuit en ligne)
- *Deep Reinforcement Learning Hands-On* — Maxim Lapan (approche pratique avec PyTorch)

## Outils
- [Gymnasium](https://gymnasium.farama.org/) — environnements de simulation (successeur d'OpenAI Gym)
- [Stable-Baselines3](https://stable-baselines3.readthedocs.io/) — implémentations fiables d'algorithmes RL
- [CleanRL](https://github.com/vwxyzjn/cleanrl) — implémentations single-file pour comprendre les algorithmes
- [PettingZoo](https://pettingzoo.farama.org/) — environnements multi-agents
