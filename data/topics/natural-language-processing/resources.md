# Ressources — Natural Language Processing

## Cours en ligne
- [Stanford CS224n — NLP with Deep Learning](https://web.stanford.edu/class/cs224n/) — le cours de référence
- [Hugging Face NLP Course](https://huggingface.co/learn/nlp-course) — gratuit, très pratique
- [fast.ai — NLP](https://www.fast.ai/) — approche pratique

## Livres
- *Speech and Language Processing* — Dan Jurafsky, James H. Martin (gratuit en ligne, le livre de référence)
- *Natural Language Processing with Transformers* — Lewis Tunstall et al. (O'Reilly, pratique avec Hugging Face)

## Outils
- [Hugging Face Transformers](https://huggingface.co/docs/transformers) — bibliothèque de référence pour les modèles NLP
- [spaCy](https://spacy.io/) — NLP industriel (NER, parsing, classification)
- [NLTK](https://www.nltk.org/) — toolkit classique pour le NLP (pédagogique)
- [LangChain](https://www.langchain.com/) — orchestration d'applications LLM
- [Ollama](https://ollama.ai/) — LLMs en local
